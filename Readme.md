# Where Winds Meet – Buff Data Dump (Learning Purpose)

This project is created **strictly for learning and research purposes**.

The data dumping and script analysis performed here are intended to:
- Study Lua runtime behavior in Unreal Engine–based games
- Learn reverse-engineering techniques for scripting systems
- Understand localization and data-driven game design
- Practice tooling, automation, and data extraction workflows

⚠️ **Disclaimer**
- This project is **not intended for cheating, exploitation, or bypassing game protections**
- No modification of game logic, server communication, or competitive behavior is involved
- All scripts operate in a **read-only / observational manner**
- Dumped data is used **solely for educational analysis**
